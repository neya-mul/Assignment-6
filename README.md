The name of this project is "DigiTools Platforms"


I have created this project as part of my web course "AI Powered Future Ready Web Dev" by Programming Hero Batch-13 and it was my Assignment 6. This project is like an e-commerce website but no products are sold here. There are some services which I can buy from this website.At the top there is a navbar. A shopping cart icon has been added into the navbar.  There are some cards has shown as services. Each card has a button and after clicking the button, that service will be added to the cart and the total price will be shown in the cart section. There are two buttons, one is for products and another is for the cart. The number of the elements added into the cart will be shown at the cart button and also the navbar cart icon.The total price will be shown in the cart section. You can also delete a service from the cart and if you delete any service the price will decrease.


To make this project, I used HTML5, Tailwind Css, DaisyUI, Javascript ES6 and React.js.


There are so many features in this project. I am telling you about 3 of them:

 1-Every card has a button. After clicking the button the button text will change and that service will be added into the cart section.

 2-In the cart section, every service has a delete button and after clicking the delete button that service will be removed from the cart.

 3-In the cart section, there is a pricing section in which the total price will be shown and the price will be change dynamically. Adding a servic will increase the price and after removing a service the amount of that service will be subtracked from the total price.
