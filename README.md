The name of this project is "DigiTools Platforms"


I have created this project as a part of my web course "AI Powered Future Ready Web Dev" by Programming Hero Batch-13 and it is my assignment 6. This project is like an e-commerce website but no product is sold here. There are some services which I could buy from here.There is a pricing section , there are some cards has shown as a service. Each card has a button and after clicking the button that service will be added to the cart and the total price will be shown there. You can also delete a service from the cart and if you delete any service the price will be decrease.


To make this project , I used HTML5, Tailwind Css, DaisyUI, Javascript ES6, React.js.


There are so many features in this project:

 1-Every card has a button, after clicking the button the button text will change and that service will me added into the cart section.

 2-In the cart section, Every service has a delete button and after clicking the delete button that service will be removed from the cart.

 3-In the cart section, there is a pricing section in which total price will be shown and the price will be change dynamically. Adding service will increase the price and after remove a service the amount of that service will be deleted from the total price.
